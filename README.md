# Blinkit Sales Data Analysis

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on a Blinkit dataset to understand sales patterns, product distribution, and outlet performance.

The notebook analyzes different factors affecting product sales such as:

* Product categories
* Outlet types
* Outlet establishment year
* Location tiers
* Fat content of products

The goal is to extract **useful business insights** from the dataset using Python-based data analysis tools.

## Analysis Performed

The notebook includes:

* Data loading and inspection
* Data cleaning and preprocessing
* Exploratory data analysis
* Sales distribution analysis
* Outlet performance analysis
* Product category insights
* Visualization of trends and patterns

---

## Key Insights

Some of the insights derived from the analysis include:

* Sales variation across different outlet types
* Performance comparison of Tier 1, Tier 2, and Tier 3 locations
* Impact of product fat content on sales
* Relationship between outlet establishment year and sales
